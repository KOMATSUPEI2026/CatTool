# 校譯台（cat-tool-demo）第 35 版 網站地圖

> 單頁應用（SPA），所有區域在同一 HTML 內以分頁切換，無真實路由。

```
校譯台（單頁應用）
│
├── 全域 Header（所有分頁常駐）
│   ├── Logo「校」+ 標題 + 副標
│   ├── 統計列：術語條目 / 記憶句段 / 文件數
│   ├── 🌙 暗黑模式切換鈕（bi-moon / bi-sun）
│   └── 🔍 防老花模式切換鈕（bi-zoom-in，1x/1.2x/1.4x）
│
├── 分頁導覽列（5 分頁）
│
├── 快速置頂/置底膠囊（fixed，貼內容區右緣、垂直置中）
│   └── 僅在 ②專案管理區、③翻譯工作區 顯示；bi-caret-up / bi-caret-down
│
├── ① 入稿工作區（預設起始頁）
│   ├── 語系配對列（防呆閘門）
│   │   ├── 來源語系 select（14 語系）→ 目標語系 select
│   │   └── 狀態提示：未選/相同語系=封鎖入稿；配對完成=teal 顯示配對
│   ├── 貼上入稿卡
│   │   ├── 文件名稱（留空自動命名；重名自動加 (2)）
│   │   ├── 原文貼上區（textarea）
│   │   └── 「建立檔案」（依「。！？＋換行」分段；未配對語系時 disabled）
│   └── xlsx 拖放入稿卡
│       ├── 前綴欄位（前綴-分頁名；留空用分頁名）
│       ├── 拖放區（拖入或點擊選檔 → 解析摘要預覽 → 可重拖更換）
│       │   └── 表頭依所選語系代碼定位欄位＋標號→srcNo；空白模板列略過
│       └── 「建立檔案」（一檔多分頁一次建立；語系變更即作廢暫存）
│
├── ② 專案管理區
│   ├── 檔名搜尋框、「+ 新增資料夾」
│   └── 檔案表格
│       ├── 欄位：文件名稱|語言(srcLang▶tgtLang)|原文字數|譯文字數|翻譯進度|校對進度|建立時間|更新時間
│       ├── 資料夾列（收合/展開、小計、✕ 刪除→檔案回未分類）
│       └── 每列：點檔名開啟(→③)|資料夾下拉|✕ 刪除檔案
│
├── ③ 翻譯工作區（核心作業區）
│   ├── 情境列（單卡整合）
│   │   ├── 目前檔案名 +「← 返回專案管理區」「⟲ 重置確認狀態」
│   │   └── 翻譯進度條（朱紅淺）＋校對進度條（teal 淺），即時連動

│   ├── 句段卡片列表

│       ├── Tab B 搜尋翻譯記憶

│
└── ⑤ 翻譯記憶
    ├── 搜尋框（原文或譯文）、「匯出 JSON」「匯入 JSON」
    ├── 表格：原文|譯文|語言(srcLang▶tgtLang)|出處|✕
    │   （刪除→對應句段徽章退空心、譯文保留）
    └── 分頁列（每頁 10 筆）
```

## 分頁間資料流向

```
入稿（語系配對閘門）──建立檔案(帶srcLang/tgtLang)──▶ 專案管理區 ──開啟──▶ 翻譯工作區
                                                                    │
        術語庫 ◀──反白新增/卡片編輯/刪除（雙向連動；嚴格同配對比對）──┤
                                                                    │
        翻譯記憶 ◀──Tab確認存入（單向；帶文件配對）───────────────────┘
                └──側欄（僅同配對）：相似/搜尋，Tab更新、Enter套用──▶ 翻譯工作區
                                                （套用即編輯→退回未確認）
```

## Modal 清單（疊加層）

| Modal | 觸發位置 | 寬度 | 按鈕 |
|---|---|---|---|
| 新增術語 | 反白原文 →「+新增術語」 | 360 | 取消/新增 |
| 編輯術語 | 術語卡片 △ | 360 | 取消/儲存 |
| 刪除術語（置中） | 術語卡片 ✕ | 360 | 取消刪除/確定刪除 |
| 取代譯文（置中） | 搜尋取代列「取代」 | 360 | 取消取代/確定取代 |
| 重置確認狀態（置中） | 情境列「⟲ 重置確認狀態」 | 440（加寬） | 取消重置/確定重置 |

## 匯出/匯入 JSON 格式（V34 起）

- 鍵名依實際語系動態產生：`{"en": "...", "zh-TW": "...", "confirmed": true, "source": "O (2)", "srcLang": "en", "tgtLang": "zh-TW"}`
- 匯入相容四種歷史格式（動態鍵/舊 ja+zh 帶標記/最舊無標記/V33 過渡格式）
