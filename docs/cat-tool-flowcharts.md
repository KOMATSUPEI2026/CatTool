# 校譯台（cat-tool-demo）第 35 版 五大區塊流程圖

> 流程圖使用 Mermaid 語法，可在支援 Mermaid 的檢視器（GitHub、Notion、VS Code 等）直接渲染；每張圖附文字說明。

---

## ① 入稿工作區

```mermaid
flowchart TD
    A[進入入稿工作區] --> B{語系配對閘門}
    B -->|未選或相同語系| C[兩條入稿路徑封鎖<br>建立檔案 disabled]
    B -->|配對完成| D[提示轉 teal<br>顯示 來源→目標]
    D --> E{選擇入稿方式}
    E -->|貼上入稿| F[輸入文件名稱<br>貼上原文]
    F --> G[建立檔案：依「。！？」與換行分段<br>重名自動改為 名稱 -2-]
    E -->|xlsx 拖放| H[拖入或點選 .xlsx]
    H --> I[逐分頁解析：表頭依語系代碼定位欄位<br>標號→srcNo；空白模板列略過<br>CRLF→LF]
    I --> V{表頭雙欄必備<br>來源欄＋目標欄}
    V -->|全部分頁不符| W[置中 Modal 報錯<br>列所選配對 vs 偵測到的語系欄<br>拖放區回初始狀態]
    V -->|至少一分頁符合| J[摘要預覽：檔名/分頁數/各頁段數<br>不符分頁註記略過；可重拖更換]
    J --> K[建立檔案：前綴-分頁名<br>每分頁一文件；譯文帶入但一律未確認]
    G & K --> L[文件記錄 srcLang/tgtLang<br>自動跳轉專案管理區]
    M[語系配對中途變更] -.->|作廢暫存 xlsx| H
```

**說明**：語系配對是唯一入口閘門。xlsx 為兩段式（解析→預覽→建檔），解析失敗不留垃圾。表頭雙欄必備：來源與目標語系欄缺一即擋該分頁，全部分頁不符時彈置中 Modal 報錯（防止選錯配對無聲匯入）。匯入的譯文照填但徽章空心——進 TM 的唯一入口仍是 Tab。

## ② 專案管理區

```mermaid
flowchart TD
    A[進入專案管理區<br>分頁籤切入即重繪] --> B[表格：語言配對/字數<br>翻譯進度%/校對進度%]
    B --> C{使用者操作}
    C -->|檔名搜尋| D[即時過濾]
    C -->|+ 新增資料夾| E[建立資料夾列]
    C -->|下拉| F[檔案指派資料夾/未分類]
    C -->|✕ 刪資料夾| G[檔案移回未分類]
    C -->|✕ 刪檔案| H[檔案與句段刪除<br>TM 紀錄保留]
    C -->|點文件名稱| I[設為目前檔案→翻譯工作區]
```

**說明**：翻譯進度% = 有譯文的句段比例（含草稿）；校對進度% = Tab 確認比例。兩值與工作區進度條同源（docStats）。

## ③ 翻譯工作區

### 主流程（含 V28 核心規則）

```mermaid
flowchart TD
    A[開啟檔案] --> B[句段卡片：原文含同配對術語標記<br>譯文框自動撐高]
    B --> C[輸入譯文<br>翻譯進度條即時上升]
    C --> D{按 Tab}
    D -->|有內容| E[存入 TM 帶文件語系配對<br>有 tmId 覆寫同筆/無則新增<br>徽章實心 teal；校對進度上升]
    D -->|空| F[退回未確認、解除 tmId<br>TM 紀錄保留]
    E --> G[焦點移至下一句]
    H[已確認句段被編輯<br>打字/清空/Enter套用] -->|立即| I[confirmed→false 徽章退空心<br>tmId 保留→重按Tab覆寫同筆]
```

### 重置確認狀態

```mermaid
flowchart TD
    A[⟲ 重置確認狀態] --> B{有已確認句段?}
    B -->|無| C[提示：沒有已確認句段]
    B -->|有| D[置中 Modal 440寬<br>N 句將退回未確認]
    D -->|確定| E[全部 confirmed→false<br>tmId 保留；譯文與TM不動<br>作廢取代復原快照]
```

### 搜尋取代（不變）+ 側欄

```mermaid
flowchart TD
    subgraph 右側欄翻譯記憶（嚴格同配對）
    A[相似模式：聚焦句段<br>bigram Jaccard 前8筆] 
    B[搜尋模式：關鍵字]
    A & B -->|只列 samePair 紀錄| C[卡片譯文欄]
    C -->|Tab| D[更新該筆記憶]
    C -->|Enter| E[套用至左側句段<br>該句退回未確認]
    end
    subgraph 左側欄頁面檢視
    F[上下頁：前/後一檔整篇唯讀]
    G[跨頁檢視：檔名/語系代碼/內文<br>三路搜尋，最多列3檔]
    end
```

## ④ 術語庫 / ⑤ 翻譯記憶（共同模式）

```mermaid
flowchart TD
    A[進入分頁] --> B[表格含語言配對欄<br>每頁 10 筆 + 頁碼列]
    B --> C{操作}
    C -->|搜尋| D[即時過濾＋回第1頁]
    C -->|頁碼點擊| E[換頁；刪除後自動夾回]
    C -->|匯出 JSON| F[動態鍵名格式<br>en/zh-TW 依實際語系]
    C -->|匯入 JSON| G[相容四種歷史格式<br>缺語系欄補 ja→zh-TW]
    C -->|✕ 刪除 TM| H[掃描解除句段 tmId<br>徽章退空心/譯文保留<br>校對進度下修]
```

**說明**：管理頁顯示全部配對的紀錄（跨池管理用）；工作區的比對嚴格限同配對。

## 跨區塊資料關係總覽

```mermaid
flowchart LR
    A[① 入稿<br>語系閘門] -->|建檔 帶srcLang/tgtLang| B[② 專案管理]
    B -->|開啟| C[③ 翻譯工作區]
    C <-->|雙向連動<br>嚴格同配對| D[④ 術語庫]
    C -->|Tab 確認寫入<br>帶配對標記| E[⑤ 翻譯記憶]
    E -.->|刪除→徽章退回<br>譯文保留| C
    E -->|側欄僅同配對<br>Enter套用→退回未確認| C
    C -->|docStats 進度%| B
```
